# shrineprayservice

## 概要

Spring Bootを使用した神社向け祈願予約管理アプリです。

神社の祈願予約をWebフォームから受け付け、入力内容をPostgreSQLに保存し、職員・管理者が予約内容を確認、編集、管理できることを目的としています。

現在開発中です。

## 使用技術

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- PostgreSQL
- HTML / CSS
- Git / GitHub

## 業務背景

神社で祈願を行う際は、祝詞で読み上げるため、事前に以下のような情報を把握しておく必要があります。

- 氏名
- ふりがな
- 住所
- 住所のふりがな（祝詞で読み上げる際の誤読防止）
- 生年月日（和暦）
- 願意
- 性別（厄年判断・初宮詣などで祝詞の内容が変わるため）
- 初宮詣では両親の名前、会社祈願では会社名など、祈願内容に応じた追加情報

一般的な来店予約とは異なり、祈願内容や神社ごとの運用に応じて必要な情報が変わる点が特徴です。

## 願意について

願意とは、祈願における願い事のことです。

現在の区分は以下を想定しています。

- 無病息災
- 恋愛成就
- 家内安全
- 商売繁盛
- 学業成就
- 交通安全
- 開運厄除
- 子宝祈願
- 結婚祈願
- 長寿祈願
- その他

神社ごとの運用や特色に合わせてカスタマイズできる構成を意識して作成しています。

## 学習目的

このアプリは、オブジェクト指向言語の基本的なWebアプリ開発を学習するために作成しています。
